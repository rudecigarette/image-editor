# 图像编辑器

一个基于 Vue 3 的现代化图像编辑 Web 应用。

## ✨ 功能特性

- 📤 **图片上传** - 支持 JPG、PNG、WebP 等格式
- ✂️ **智能裁剪** - 可视化裁剪框，支持毫米/厘米单位
- 🔍 **缩放控制** - 精确的缩放和移动功能
- 🎯 **DPI 设置** - 自定义输出 DPI（72-600）
- 🔄 **格式转换** - 支持 PNG、JPEG、WebP 格式
- ↶↷ **撤回前进** - 完整的历史记录管理
- 📱 **移动端适配** - 完美支持触摸操作
- 💾 **倍数导出** - 支持 1x-5x 倍数导出

## 🚀 在线使用

直接访问：[图像编辑器](https://你的用户名.github.io/image-editor/image-editor-standalone.html)

## 💻 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 生成单文件HTML
node build-standalone.js
```

## 📦 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- Vite - 下一代前端构建工具
- Canvas API - 原生图像处理
